コメントピックアップアプリ

概要: 講義に対して寄せられた大量のアンケートについて、AIがコメントを分析し、フィードバックを与えるアプリです。 すべてのアンケートに人の力で目を通すのが困難であり、講義の改善に生かせていないという課題を改善することがこのアプリの目的です。 ユーザーはアンケート結果を含むcsvファイルをアップロードすると、アンケート内のポジティブ、ネガティブコメントを要約し、 講義改善につながる重要なコメントや、危険度の高いコメント(誹謗中傷や法に抵触するようなコメント)を抽出します。 また、アンケート全体の傾向をグラフで可視化します。

使用技術:

  • フロントエンド: HTML, CSS, JavaScript
  • バックエンド: Python, FastAPI
  • AIモデル: Google Gemini API
  • CI/CD GitHub Actions, Render
